Updates for August 1, 2023

Public Beta, Video Support & more

Raster is live on ProductHunt and is now in Public beta.

NEW! Raster now supports video hosting

Raster now supports video. No editing functionality at the moment, but you can store, organize, tag and serve MP4 files via the Raster CDN.

Improvements

  • New Metadata: "Uploaded By"
  • Click to copy photo dimensions
  • If you invite someone to an organization who hasn't joined Raster Private Beta yet, they will also receive an invite to Raster
  • When creating a new version of a photo, Raster will not intelligently name the version based off of the branching version name, rather than incrementing "Untitled"

  • Organization names can now be three or more characters (previously required four characters)

Bug Fixes

  • Fixed an issue where version names may not save correctly
  • Fixed an issue where photo dimensions may not display properly
  • Fixed an instance where Empty trash would not work
  • Fixed multiple small issues with dragging and scrolling
  • Some dark mode text color fixes
