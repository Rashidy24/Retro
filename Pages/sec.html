Updates for March 7, 2024

Stability & Raster Image Stack

We've completely rewritten the back-end of Raster to a custom stack. What this means is that we'll finally be able to deliver on the following items today:

  • Instant edit previews, no more waiting several seconds for image previews to update
  • Shorter URLs
  • Faster CDN image load times throughout And the following features in the near future:
  • Replace versions that have already been published
  • AI-based editing features
  • More edits and other cool stuff

This is a huge update and there may be bugs and issues in the beginning, but it enables us to finally take Raster to where we envision it can be.

Improvements & Bug Fixes

  • Improved tag design
  • Bumped dependencies & security fixes
  • Improved image tags using Google Gemini
