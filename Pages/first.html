Updates for July 4, 2024

Big Things Ahead

We've learned a great deal in the past few months about scaling the back-end of Raster. With larger libraries demanding more from Raster's front and back-end, we've identified and been working on key opportunities to improve the user experience.

From rewriting the image edit stack to database overhaul, we have big things in store for what Raster can do.

New Features & Improvements

  • Improved quality of AI tagging once again
  • NEW: Full RAW image support
  • Soon: Google Drive import coming to Enterprise customers
  • Raster now supports M4V video format as well

Bug Fixes

  • Fixed an annoying issue where Raster would silently fail on image upload. This was due to unknown, large, or corrupted metadata. Raster now nicely lets you know what's going on, so you're not left in the dark again
  • Improved conversion of original files to Raster original formats
  • Improved back-end processes for deleting user data. When you delete a library, we make sure all your images are actually deleted on the server as well.
  • Fixed issues where metadata transfer would silently fail
